=============
 About Pungi
=============

Pungi is a distribution compose tool.

Composes are release snapshots that contain release deliverables such as:

- installation trees

  - RPMs
  - repodata
  - comps

- (bootable) ISOs
- kickstart trees

  - anaconda images
  - images for PXE boot
